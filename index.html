Info materia

Nombre: Desarrollo Web I

Profesor: Guillermo Calderon

Descripción: Es una asignatura práctica que introduce los fundamentos de la creación de sitios y aplicaciones web mediante HTML, CSS y JavaScript. Permite desarrollar competencias en diseño, maquetación e interactividad web, comprendiendo la arquitectura del front-end y aplicando buenas prácticas de programación y diseño responsivo.

Guías

Guía 1 — HTML básico

Construcción de páginas semánticas, tablas informativas y maquetación inicial. Se ponen en práctica encabezados jerárquicos, listas descriptivas, enlaces y recursos multimedia, destacando ejemplos como países, series y estilos embebidos que refuerzan la consistencia de estructura.

  • Ejemplo 2
  • Ejemplo Países
  • Ejemplo Serie

Guía 2 — CSS y Flexbox

Uso de Flexbox para organizar biografías, catálogos y tableros, reforzando jerarquía visual y selección de tipografías. Se enfatiza la creación de contenedores alineados, tarjetas con sombras y combinaciones de color coherentes para producir composiciones limpias y equilibradas.

  • Mi Biografía
  • Películas
  • Tablero

Guía 3 — Grid Layout

Aplicación de CSS Grid para crear cuadrículas responsivas, control de columnas y filas, y superposición de capas con efectos visuales. El trabajo abarca agencias de viaje, menús horizontales y estudios de color que demuestran la versatilidad del layout en diferentes contextos.

  • Nominaciones Oscares
  • Ejemplo Z-index
  • Agencia de viajes
  • Ejemplo menu horizontal

Guía 4 — Responsive design

Implementación avanzada de media queries, componentes adaptables y tipografías fluidas. Cada página incluye versiones diferenciadas para dispositivos móviles, tabletas y escritorio, junto con un sistema de estilos base que garantiza continuidad visual sin importar el ancho de la pantalla.

  • Landing responsiva
  • Media queries
